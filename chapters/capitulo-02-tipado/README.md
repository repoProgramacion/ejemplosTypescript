# Capítulo 2. Sistema de Tipado

Este capítulo explica cómo TypeScript modela datos con tipos explícitos e inferidos.

## Idea central

El sistema de tipos detecta errores antes de ejecutar el programa. Eso reduce fallos y mejora la lectura del código.

## Qué practica el ejemplo

- Tipos primitivos.
- `type` aliases.
- `union types`.
- `unknown` frente a validación segura.

## Cómo ejecutarlo

```bash
bun run chapters/capitulo-02-tipado/ejemplo.ts
```
