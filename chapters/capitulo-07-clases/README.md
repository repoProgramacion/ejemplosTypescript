# Capítulo 7. Clases

Este capítulo trabaja programación orientada a objetos con TypeScript moderno.

## Idea central

Las clases permiten agrupar datos y comportamiento. TypeScript añade encapsulación, `readonly`, herencia y abstracción con seguridad.

## Qué practica este capítulo

- Constructor abreviado.
- Métodos públicos y privados.
- Herencia con `extends`.
- Clases abstractas.
- Getters y setters.
- `readonly`, `private` y `protected`.
