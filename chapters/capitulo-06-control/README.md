# Capítulo 6. Estructuras de Control

Este capítulo enseña decisiones, repetición y el patrón de uniones discriminadas.

## Idea central

Las estructuras de control organizan el flujo del programa. TypeScript además refina tipos dentro de cada rama.

## Qué practica este capítulo

- `if`.
- `for...of`.
- `switch`.
- `never` para control exhaustivo.
- `while`.
- `do...while`.
- Operador ternario.
- `for` clásico y `for...in`.
