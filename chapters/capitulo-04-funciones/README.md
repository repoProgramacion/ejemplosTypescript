# Capítulo 4. Funciones

Este capítulo presenta funciones tradicionales con parámetros, valores por defecto y retorno tipado.

## Idea central

Una función es una unidad reutilizable que recibe datos, aplica reglas y retorna un resultado confiable.

## Qué practica el ejemplo

- Parámetros tipados.
- Valores por defecto.
- Parámetro rest.
- Funciones como tipos.

## Cómo ejecutarlo

```bash
bun run chapters/capitulo-04-funciones/ejemplo.ts
```
