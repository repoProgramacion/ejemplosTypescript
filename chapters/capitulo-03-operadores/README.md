# Capítulo 3. Operadores

Este capítulo reúne operadores aritméticos, relacionales y modernos de TypeScript.

## Idea central

Los operadores no solo transforman valores. También modelan intención, especialmente con `??` y `?.`.

## Qué practica el ejemplo

- Suma, promedio y comparación.
- `optional chaining`.
- `nullish coalescing`.
- Operador `satisfies`.

## Cómo ejecutarlo

```bash
bun run chapters/capitulo-03-operadores/ejemplo.ts
```
