# Capítulo 1. Ecosistema TypeScript y Bun

Este capítulo introduce la relación entre JavaScript, TypeScript y Bun.

## Idea central

TypeScript añade verificación estática sobre JavaScript. Bun actúa como runtime, gestor de paquetes y ejecutor rápido de archivos TypeScript.

## Qué practica este capítulo

- Declaración de constantes con tipos inferidos.
- Ejecución directa de `.ts` con Bun.
- Uso básico de `console.log`.
- Lectura conceptual de una configuración moderna.
- Scripts con Bun.
- Módulos ESM en TypeScript.

## Archivos incluidos

```bash
bun run chapters/capitulo-01-ecosistema/ejemplo.ts
bun run chapters/capitulo-01-ecosistema/scripts-bun.ts
bun run chapters/capitulo-01-ecosistema/modulos.ts
```
