# Capítulo 5. Funciones Flecha

Este capítulo muestra la sintaxis moderna de funciones y el valor práctico del `this` léxico.

## Idea central

Las arrow functions son compactas y muy útiles en callbacks, transformaciones de arreglos y cierres.

## Qué practica el ejemplo

- Sintaxis flecha.
- `map`, `filter` y `reduce`.
- Cierre léxico.

## Cómo ejecutarlo

```bash
bun run chapters/capitulo-05-funciones-flecha/ejemplo.ts
```
